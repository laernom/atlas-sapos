Exportar HTML Actualizado

Este botón genera un nuevo atlas-william.html con todos tus cambios integrados en el código. Ese es el archivo que subes a GitHub Pages.

📖 Cómo publicar en GitHub Pages
Paso 1. Crea una cuenta en github.com si no tienes.
Paso 2. Crea un nuevo repositorio público. Llámalo por ejemplo atlas-william.
Paso 3. Sube el HTML descargado renombrándolo a index.html.
Paso 4. Ve a Settings → Pages del repositorio. En Source selecciona main / root y guarda.
Paso 5. En unos minutos tendrás tu atlas en https://tunombredeusuario.github.io/atlas-william/

Para actualizar: cada vez que hagas cambios en el panel admin, descarga el HTML de nuevo y reemplaza el index.html en GitHub.

☁ Sincronizar con GitHub

Publica tus cambios directamente en GitHub Pages sin descargar ni subir nada a mano. Solo necesitas un token de acceso personal de GitHub.

El token solo se guarda en memoria mientras la pestaña está abierta. Al cerrar el navegador desaparece y tendrás que introducirlo de nuevo.

📖 Cómo crear el token
1. Ve a github.com/settings/tokens/new
2. Escribe una nota: Atlas William
3. Marca solo el permiso repo (acceso completo a repositorios)
4. Haz clic en Generate token y cópialo aquí
5. El repositorio debe existir ya en GitHub con Pages activado

Cada vez que pulses Publicar ahora, el atlas público se actualiza en segundos.
